## installation PIP requise

````
pip install beautifulsoup4

pip install PyInquirer
````

## Règle du jeu

````
Pour lancer le jeu 3 options s'offre à vous :️

1) lancer le jeu avec des url au hasard avec la commande :️

   python3 realGame.py

2) Générer une "graine" pour le mode duel avec la commande :
️
   python3 realGame.py -n

3) Lance une partie avec une "graine" avec la commande :
️
   python3 realGame.py -s 'premiereURL#deuxiemeURL'

Exemple : 

python3 realGame.py -s 'Andasta_benoiti#Henri_Durville'

Vous pouvez également copier coller la "graine quand elle est générée
````
